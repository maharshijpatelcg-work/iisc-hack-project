## Overview

AI Adaptive Onboarding Engine is an intelligent platform that analyzes resumes, matches job requirements, detects skill gaps, and generates a customized onboarding roadmap.

This system automates and enhances the onboarding process by:

- Analyzing candidate resumes  
- Comparing with job requirements  
- Identifying missing skills  
- Generating personalized learning paths

## Installation & Setup

### Clone Repository

git clone https://github.com/Ritesh007-max/iisc-hack-project.git  
cd iisc-hack-project/ai-onboarding-engine  

---

### Run Frontend

cd frontend  
npm install  
npm run dev  

---

### Run Backend

cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload  

---

## How It Works

1. Upload your resume  
2. Add job description  
3. Enter career goals (optional)  
4. Click "Generate Analysis"  
5. Get:
   - Extracted skills  
   - Skill gap insights  
   - Personalized roadmap
